# Enes Celal Yavuz - Gereksinim Analizi

Bu dosyada, FluentBee (İngilizce Öğrenme Platformu) projesinde şahsım tarafından üstlenilmiş olan fonksiyonel gereksinimler yer almaktadır. Tüm gereksinimler, eylem bildiren isimlerle adlandırılmış ve teknik jargon içermeyen tanımlamalarla açıklanmıştır.

- **Öğrenci Kayıt Olma** (POST API Metodu)
  - **Açıklama:** Sisteme yeni bir kullanıcı hesabının oluşturulması.
- **Profil Güncelleme** (PUT API Metodu)
  - **Açıklama:** Kullanıcının kendi ad, soyad ve şifre gibi kişisel bilgilerini değiştirmesi.
- **Kelimeleri Listeleme** (GET API Metodu)
  - **Açıklama:** Sistemdeki tüm İngilizce kelimelerin Türkçe anlamlarıyla birlikte liste halinde ekrana getirilmesi.
- **Kelime Silme** (DELETE API Metodu)
  - **Açıklama:** Kullanıcının kendi özel çalışma listesinden artık istemediği kelimeleri çıkarması.
- **Dersleri Listeleme** (GET API Metodu)
  - **Açıklama:** İngilizce seviyelerine göre hazırlanmış ders modüllerinin seviye bazlı gösterilmesi.
- **Çalışma Hedefi Güncelleme** (PUT API Metodu)
  - **Açıklama:** Kullanıcının uygulama üzerinde belirlediği günlük hedef çalışma süresini yeniden ayarlaması.
- **Favori Kelime Ekleme** (POST API Metodu)
  - **Açıklama:** Öğrencinin çalışma listesine yepyeni bir İngilizce kelime kaydetmesi.
- **Öğrenme İstatistiklerini Görme** (GET API Metodu)
  - **Açıklama:** Öğrencinin kaç kelime ezberlediğine ve hangi dersleri bitirdiğine dair verilerin sunulması.
